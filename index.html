Полное руководство по игре draword

1. Цель игры

Составьте правильные слова из перемешанных букв на игровом поле. В каждом уровне нужно собрать несколько слов.

2. Основные правила

  1. Каждая строка - отдельное слово
  2. Кликайте по буквам последовательно, чтобы поменять их местами
  3. Темные плитки (фиксированные) нельзя перемещать
  4. Когда все индикаторы станут зелеными - вы победили!

3. Выбор сложности

  • Легкий: 3 слова по 3 буквы
  • Средний: 4 слова по 4 буквы (1 фиксированная буква)
  • Сложный: 5 слов по 5 букв (2 фиксированные буквы)

4. Подсказки

Используйте кнопку "Подсказка", затем выберите плитку - игра покажет правильную букву и зафиксирует ее.

5. Ответы

Кнопка "Ответы" покажет все правильные слова с ссылками на Википедию.

6. Поделиться результатом

После победы вы можете поделиться скриншотом своего результата в соцсетях.

7. Темы оформления

Измените внешний вид игры через панель тем в правом нижнем углу.
